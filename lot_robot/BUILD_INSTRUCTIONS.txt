==========================================
ИНСТРУКЦИЯ ПО СБОРКЕ ПРИЛОЖЕНИЯ
==========================================

ШАГ 1: Откройте командную строку в папке проекта
   - Нажмите Shift + Правый клик в папке проекта
   - Выберите "Открыть окно PowerShell здесь" или "Открыть в терминале"

ШАГ 2: Запустите один из скриптов сборки:

   Вариант А (простой):
      build_simple.bat

   Вариант Б (с настройками):
      build.bat

ШАГ 3: Дождитесь завершения сборки (может занять 2-5 минут)

ШАГ 4: Найдите готовый файл в папке "dist"
   - Файл: "Поиск закупок.exe"
   - Этот файл можно запускать на любом компьютере с Windows

ПРИМЕЧАНИЯ:
- При первом запуске антивирус может выдать предупреждение
  Это нормально - добавьте файл в исключения
- Файл не требует установки Python на целевом компьютере
- Можно создать ярлык на рабочем столе

ПРОБЛЕМЫ?
- Убедитесь, что Python установлен: python --version
- Проверьте, что все зависимости установлены: pip install -r requirements.txt
- Подробная информация в файле README_BUILD.md
