# Инструкция по сборке десктопного приложения

## Требования
- Python 3.8 или выше
- Windows 10/11
- Интернет-соединение (для скачивания зависимостей)

## Способ 1: Простая сборка (рекомендуется)

1. Откройте командную строку (cmd) в папке проекта
2. Запустите файл `build_simple.bat`
3. После завершения сборки исполняемый файл будет в папке `dist`

## Способ 2: Продвинутая сборка (с использованием .spec файла)

1. Откройте командную строку (cmd) в папке проекта
2. Запустите файл `build.bat`
3. Этот способ использует файл `build.spec` для более точной настройки

## Ручная сборка

Если автоматические скрипты не работают, выполните команды вручную:

```batch
# Установка PyInstaller
pip install pyinstaller

# Установка зависимостей
pip install -r requirements.txt

# Простая сборка
pyinstaller --name="Поиск закупок" --onefile --windowed --add-data "config.py;." main.py

# Или используйте .spec файл для более точной настройки
pyinstaller build.spec
```

## Распространение

После сборки в папке `dist` будет создан один исполняемый файл `Поиск закупок.exe`. 
Этот файл можно:
- Скопировать на другой компьютер (с Windows)
- Создать ярлык на рабочем столе
- Запускать напрямую

**Примечание:** При первом запуске антивирус может предупредить о неизвестном файле. 
Это нормально для скомпилированных Python-приложений. Добавьте файл в исключения антивируса.

## Устранение проблем

### Приложение не запускается
1. Убедитесь, что на компьютере установлен Visual C++ Redistributable
2. Проверьте, что файл не заблокирован антивирусом
3. Попробуйте собрать с флагом `--console` для просмотра ошибок

### Ошибки при сборке
1. Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
2. Проверьте версию Python: `python --version`
3. Попробуйте обновить PyInstaller: `pip install --upgrade pyinstaller`
